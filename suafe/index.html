Suafe (Subversion user authorization file editor) is a pair of open-source projects that together make it easiser to parse, generate and edit Subversion authorization (authz) files. Suafe API provides a Java API for editing authz files. The Suafe project is a GUI application that uses the Suafe API for editing authz files. Suafe also includes a complete command-line interface for scripting edits to authz files.

Suafe source is hosted at GitHub and built using Travis CI:

  • Suafe: GitHub, Travis CI
  • Suafe API: GitHub, Travis CI
